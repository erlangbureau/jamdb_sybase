JamDB Sybase
===============

JamDB Sybase is a small and fast driver for Sybase ASE writen in Erlang.

Goals
-----

* No third-party dependencies.
* No parameterized module.
* No process dictionary.
* No ports.
* No NIF's.
* All code written exclusively in Erlang.

Getting Started
-----
